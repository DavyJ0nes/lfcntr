# LFCNTR (Life Counter)

[![Deploy to GitHub Pages](https://github.com/davyj0nes/lfcntr/actions/workflows/deploy.yaml/badge.svg)](https://github.com/davyj0nes/lfcntr/actions/workflows/deploy.yaml)

A simple life point tracker for Magic: The Gathering, Flesh and Blood, et al.
Mainly using it just as a simple project to learn modern typescript.

## Features

- Split-screen layout for two players
- Tap top half to increase, bottom half to decrease
- Responsive design (works in landscape and portrait)
- Works on mobile and desktop

## Demo

[Live demo](https://davyj0nes.github.io/lfcntr/)

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Tech

- TypeScript
- Vite
- Vanilla JS (no framework)
